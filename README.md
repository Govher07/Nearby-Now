# Nearby Now

A location-based event discovery app that helps users find things to do nearby.

This project was initially inspired by helping fans—especially those unable to attend FIFA 2026 matches—find local venues such as cafes, bars, or public screenings to watch games together. 

To make the idea more useful beyond a single event, it was expanded into a general event discovery platform that helps users find nearby activities and experiences at any time.

## Current Status
The project is currently in Phase 2, featuring core functionality such as event filtering, user location detection, and distance-based event display.

## Features
- View nearby events (mock data)
- Filter events by category
- Detect user location
- Calculate distance to events
- Clean and responsive UI

## Tech Stack
- Next.js
- React
- TypeScript
- Tailwind CSS

## How It Works
- The app detects the user's location using the browser
- Each event has coordinates
- Distance is calculated between the user and events
- Events display how far they are from the user
